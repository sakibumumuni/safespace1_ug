ℹ️ How Email Alerts Work
The system automatically checks each user on periodic intervals based on severity tier: Urgent (every 24h), Concern (every 48h), Watch (every 72h). When flagged, a structured email is sent to the configured recipient with the anonymous user token, mood trend, and journal excerpts. The student's identity is never included. You can also manually trigger a check using the button above.
